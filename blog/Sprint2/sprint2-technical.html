What are the best practices associated with using classes vs. ids?


Thats a really good question. First off let me start by explaining what each do briefly


A Class allows you to identify more than one element, you can then use that class to referance all the elements that have this class


An ID is can only be used on 1 element. It is mainly used for when you want to do something specific with an element and dont want to change anything else


The 4 main ideas are as follows:


  1. Keep Use of IDs and Classes to a Minimum
  2. Use Functional ID and Class Names
  3. Use Generic ID and Class Names
  4. Use Names That Are as Short as Possible But as Long as Necessary

The main idea here is that you want to make sure that you dont overuse either when you dont have to. If you where to use lots of different classes and ids it makes it harder to manage. Keep them short but long enought that you know what they are for, dont use to many. More ids and classes makes your life harder than it has to be.
